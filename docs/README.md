## 说明

为了企业内部的培训开发一个可部署在局域网内的网站项目，该网页页面会被投屏到会议室的大屏幕上，供参加会议的人查看。因此要求网站的配色、字体都符合大屏幕投屏显示的风格（可以接近于PPT的风格）。

这个网站分成公共页面、配置页面和用户页面，公共页面和配置页面适配PC屏幕即可，用户页面适配手机屏幕。管理员输入密码后进入配置页面。互动内容分成公共页面和用户页面。主持人通过手机扫描二维码签到后进入主持人互动页面，可控制公共页面加载指定的互动内容的公共页面，可在主持人互动页面上，查看所有参会人互动的情况。参会人通过手机扫描二维码签到后，可进入互动内容的用户互动页面，在用户页面上进行选择、回答等操作进行互动。互动的进展（如倒计时、结果统计等）实时显示在互动内容的公共页面上。

## 功能描述

该网站的主要功能包括：

1. 签到功能：
   - 首页显示二维码，用户可以用手机扫描二维码进行签到。用户用手机扫描二维码，弹出的页面需要适配手机屏幕，为用户分配一个随机的4位数字，同时允许用户在弹出的页面上输入姓名等信息。
   - 随着用户的签到，首页上的签到人数会实时更新。 同时首页上也留出位置能够显示公司logo、培训主题、签到指引文字等内容；
   - 弹出的页面上，也支持检查用户是否签到过。如果用户已签到，弹出提示框告知用户已签到；如果用户未签到，允许用户继续签到。
   - 弹出的页面上，支持用户选择身份（如主持人、参会人等），并将用户的身份信息保存到数据库中，用户选择主持人身份为主持人后，需要录入系统为这次培训设置的主持人密码，认证通过后，才签到为主持人。
2. 互动功能：提供互动功能，如小游戏、问答、投票、参与讨论等，方便培训过程中的互动。
   - 主持人页面可从预设的互动内容列表中，选择启动互动，则所选互动内容的公共页面加载到公共页面上。
   - 用户页面上，参会人可以在互动内容的用户互动页面上，进行选择、回答等操作进行互动。
   - 互动内容的公共页面上，会实时显示互动进展（如倒计时、结果统计等）。
3. 导航功能：提供导航功能，方便培训参与者快速找到所需的内容。
4. 配置功能：提供配置功能，方便管理员对培训主题和互动内容进行配置。
   - 管理员可创建一次培训，支持重置培训记录，导出培训结果
   - 互动内容支持从预设的互动类型中选择，支持多次选择。可单独为每个互动内容配置显示时间、互动类型等参数。

## 设计约束
 - 中文界面
 - 互动内容的公共页面需要适配大屏幕投屏显示的风格（可以接近于PPT的风格）
 - 互动的用户页面需要适配手机屏幕，操作方便。
